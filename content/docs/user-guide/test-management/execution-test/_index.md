# 执行测试

测试可以是一行简单的代码，也可以是非常复杂的问题，其中包含了很多细节、步骤、链接和与之相关的文件。当我们创建好一个测试后，理所应当的应该了解如何去执行它。执行一个测试意味着为测试运行或执行时发生的事情提供一个高级的状态，并对其进行一些后续操作（如关联缺陷等）。
