# webhook日志

## 1. 概述

webhook是在平台使用已定义模板向消息机器人发送webhook的状态回执。通过此功能，用户可以了解发送webhook的状态、数量、失败原因。

## 2. webhook日志列表

列表字段：

- 发送时间：消息发送的时间。
- **状态**：邮件发送失败或成功的状态回执。
- 失败原因：邮件发送失败的原因回执。
- 项目：触发webhook的项目。
- webhook地址：发送webhook的地址。
